## Стек технологий:
JavaScript

**Front**: React + Redux, Redux Saga, HTML, CSS

**Back**: Node.js, Express, JWT, Bcrypt, Cors

**DB**: PostgreSQL, Sequelize ORM


## Планы развития:
* Полнофункциональное внедрение оплаты

### Установка:
В директории 2 папки:
* **server/** Отвечает за back-end. 
   * cd server
   * npm i
   * npm run enit
   * npm run dev
* **client/** - front-end. 
   * cd client 
   * npm i
   * npm start
